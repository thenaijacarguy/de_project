# Contributing

This is a personal learning and portfolio project. Feedback, suggestions,
and improvements are welcome.

## How to contribute

1. Fork the repository
2. Create a feature branch: `git checkout -b feature/your-feature-name`
3. Make your changes and test them end to end
4. Commit with a clear message: `git commit -m "Add: description of change"`
5. Push and open a Pull Request against the `main` branch

## Reporting issues

Open a GitHub Issue describing:
- What you expected to happen
- What actually happened
- Your OS, Python version, and Docker version
- Any relevant error messages or logs
